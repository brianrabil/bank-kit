# Changelog

All notable changes to **Bank Kit** will be documented here. This project adheres to [Semantic Versioning](https://semver.org/) whenever possible.

---

## [0.2.0] - 2025-03-20

### Added

- **Premium Components**: Introduced advanced data visualization modules (BarChart, LineChart) for analytics dashboards.
- **Mortgage Calculator**: New multi-step wizard to handle rate selection, monthly payment calculation, and disclaimers.
- **Dark Mode**: Added Tailwind classes and theming utilities to enable a dark UI theme.

### Changed

- **Transaction List**: Improved sorting options and added pagination by default.
- **Card Component**: Updated default padding and border radius for a cleaner design.

### Fixed

- **Table Rendering**: Resolved an issue where long column names would cause horizontal overflow on mobile devices.
- **Z-Index Conflicts**: Addressed stacking context conflicts when using Dialog and Tooltip simultaneously.

---

## [0.1.1] - 2025-03-01

### Added

- **Text**: Introduced `Text md Semibold` as a new class for subtle emphasis in body content.
- **Examples Folder**: Included a `LoanApplication` example showcasing multi-step forms and real-time validations with Zod.

### Fixed

- **Display 2xl**: Corrected letter spacing from `-1%` to `-2%` for better visual balance.
- **Registry Build Script**: Resolved errors when running `registry:build` on Windows environments.

---

## [0.1.0] - 2025-02-15

Initial release of **Bank Kit**.

### Added

- **Core UI Library**: Essential components (Button, Input, Card, Dialog, etc.) built on React and Tailwind CSS.
- **Shadcn Registry**: A streamlined approach to generating, syncing, and customizing UI components.
- **Turbo Monorepo**: Organized structure to manage apps, packages, and shared configurations efficiently.
- **Documentation Site**: Basic guides for Installation, Typography, and Getting Started.
