## Overview

The **Container** component provides a flexible wrapper for layout sections.  
It centers content, enforces width constraints, and applies consistent horizontal padding (gutter).

## Best Practices

- Use **width** presets to enforce consistent breakpoints across pages.  
- Adjust **gutter** to match design spacing requirements (responsive vs. always).  
- Combine with grid or flex utilities for complex layouts.  
- Use `asChild` sparingly when you need to avoid extra wrapper DOM nodes.

## Customization

Override or extend variants by importing and using `containerVariants`:

```tsx
import { containerVariants } from '@bank-kit/ui/components/container';
import { cn } from '@/lib/utils';

export function CustomContainer(props) {
  const classNames = cn(
    containerVariants({ width: '5xl', gutter: 'always' }),
    'bg-gray-50 rounded-lg',
    props.className
  );
  return <div className={classNames}>{props.children}</div>;
}
```
