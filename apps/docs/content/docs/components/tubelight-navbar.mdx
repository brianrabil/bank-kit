## Installation

```bash
bunx --bun shadcn@latest add http://localhost:3000/r/tubelight-navbar.json
```

## Usage

```tsx
import { NavBar } from "@bank-kit/registry/tubelight-navbar/tubelight-navbar";
import { Home, User, Settings } from "lucide-react";

const items = [
  { name: "Home", url: "/", icon: Home },
  { name: "Profile", url: "/profile", icon: User },
  { name: "Settings", url: "/settings", icon: Settings },
];

<NavBar items={items} />
```

## API

### `<NavBar />` Props

| Prop       | Type         | Required | Description                                      |
|------------|--------------|----------|--------------------------------------------------|
| items      | NavItem[]    | Yes      | Array of navigation items (see below)            |
| className  | string       | No       | Additional Tailwind classes for the navbar root   |

#### `NavItem`

| Prop  | Type         | Required | Description                |
|-------|--------------|----------|----------------------------|
| name  | string       | Yes      | Display name for the tab   |
| url   | string       | Yes      | Link destination           |
| icon  | LucideIcon   | Yes      | Icon from lucide-react     |

## Features

- Responsive: mobile and desktop layouts
- Animated: spring transitions for active tab
- Accessible: keyboard and screen reader friendly
- Theming: uses shadcn, Tailwind, and lucide-react

> **Note:** The navbar is fixed to the bottom on mobile and top on desktop by default. You can customize its position and style via the `className` prop.
