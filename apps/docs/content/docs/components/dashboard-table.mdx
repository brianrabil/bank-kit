## Overview

The **DashboardTable** component provides a fully-featured, draggable and sortable table for dashboard views.  
It leverages TanStack Table for data management and `@dnd-kit` for drag-and-drop row reordering.

## Installation

```bash
npm install @bank-kit/ui
```

```tsx
import { DashboardTable } from '@bank-kit/ui/components/dashboard-table';
```

## Best Practices

- Ensure each row object has a unique `id` for drag-and-drop to work.  
- Define concise column definitions for optimal performance.  
- Use `onReorder` to persist new row order to your application state or backend.
