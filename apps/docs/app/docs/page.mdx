# Welcome to Bank Kit

Bank Kit is a comprehensive React UI kit that accelerates the creation of high-performance financial products. We offer banking-specific components, pre-built flows, and responsive templates—so you can focus on business logic instead of reinventing the same UI repeatedly.

---

## Key Features

- **Banking-Themed Components** – Account overviews, transaction lists, payment forms, and more.
- **Freemium Model** – Start free with core components, then add premium extensions for advanced requirements.
- **Tailwind CSS** – Utility-first styling allows you to quickly apply custom design themes.
- **Shadcn Registry** – Easily generate and sync components using the Shadcn toolset.
- **Turbo Monorepo** – Organized code structure for efficient development and sharing of components.
